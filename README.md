Mopent Final

Deskripsi
Proyek ini adalah website prelaunch untuk brand Mopent.
Website ini digunakan untuk menampilkan konsep awal brand sebelum rilis resmi.

Tujuan
• Mengenalkan brand ke publik lebih awal
• Menampilkan visual dan identitas brand
• Mengumpulkan minat sebelum launch

Fitur Utama
• Halaman prelaunch
• Tampilan visual brand
• Informasi singkat tentang produk
• Desain responsif untuk desktop dan mobile

Teknologi
• HTML
• CSS
• JavaScript

Cara Menjalankan
• Clone repository ini
• Buka folder proyek
• Jalankan file index.html di browser

Status Proyek
• Prelaunch
• Masih bisa berubah dan dikembangkan

Catatan
Website ini belum mewakili versi final brand.
Konten dan desain dapat berubah sebelum peluncuran resmi.
